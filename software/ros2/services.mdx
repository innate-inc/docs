Use ROS2 services for request/response operations.

## Common services

* `/mars/arm/goto_js` for joint-space arm moves
* `/mars/head/set_ai_position` for head reset/position helpers

## Quick commands

```bash
ros2 service list
ros2 service type /mars/arm/goto_js
ros2 service call /mars/head/set_ai_position std_srvs/srv/Trigger {}
```

For all service signatures and message details, see [ROS2 Reference](/software/advanced/ros2-reference).
